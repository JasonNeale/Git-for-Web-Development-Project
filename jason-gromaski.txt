1. What is Semantic HTML?
    Semantic HTML are HTML tags with descriptive amd meaningful tag names.

2. What is HTML used for?
    HTML stands for Hypertext Markup Language. It is used to define elements 
    within a web page so the web browser can output a graphical interpretation.

3. What is an attribute and where do we put it?
    An attribute provides an html tag with extra information such as classes and id style and more.

4. What is the h1 tag used for? How many times should I use it on a page?
    The h1 tag is a heading tag and should only be used once in an html page to define the page title.

5. Name two tags that have required attributes
    input, img

6. What do we put in the head of our HTML document? 
    All the tags with info we want the browser to access but not display within the web page such as title and meta tags.

7. What is an id? 
    And id attribute assigns an element a unique id that can be used by css for styling or by javascript

8. What elements can I add an id to? 
    Including the body tag and every tag that goes between.

9. How many times can I use the same id on a page? 
    Only once. all id attributes should be unique.

10. What is a class? 
    It is a selector type used to give an html element a group of style attributes.

11. What elements can I add a class to? 
    Including the body tag and every tag that goes between.

12. How many times can I use the same class on a page? 
    Class selector use can be repeated infinitely.

13. How do I get my link to open in a new tab?
    With the target attribute in the anchor tag then setting the value to "_blank"

14. What is the alt attribute used for? 
    It's used to display a text replacement for an image if it does not load

15. How do I reference an id?
    By using the pound, or hashtag, '#idName' symbol then the id name

16. What is the difference between a section and a div
    The only real difference is semantic naming of the tag otherwise they perform identically

17. What is CSS used for? 
    CSS (Cascading Style Sheet) is used to style html elements.

18. How do we select an element? Example - every h2 on the page
    Using the element selector h2{}

19. What is the difference between a class and an id? - Give me an example of when I might use each one
    The id is used to identify one element and a class can identify a single element.
    For example you have 4 divs with the class .divClass with individual ids
    <div class="divClass" id="div1"></div>
    <div class="divClass" id="div2"></div>
    <div class="divClass" id="div3"></div>
    <div class="divClass" id="div4"></div>

1. What is the difference between an inline element and a block element?
    Inline elements don't force a new line in the web page but block elements do. 
    Both can still be manipulated by CSS.

2. What happens when an element is positioned absolutely?

5. In your own words, explain the box model. What is the fix for the box model?
    It's a box that wraps around every html elements made up of margin, border, padding, content.
    Box-sizing and border-box fixes the default box model that can make closely packed elements overlap.
